Speed Math Game
A timed math quiz game with three difficulty levels (Easy, Medium, Hard) and a global leaderboard. Players solve math equations as quickly as possible, earning points based on speed and accuracy. Each difficulty level has its own leaderboard with color-coded themes.

Built with vanilla JavaScript and Supabase for real-time leaderboard functionality. The game has a countdown timer, streak bonuses, and instant feedback on answers.

To install:
1. git clone the repo with: git clone https://github.com/4ndr3jS/SpeedMath
2. Run npm install and npm run dev to start the server.
3. Make sure to configure your Supabase credentials in a .env file with VITE_SUPABASE_URL and VITE_SUPABASE_ANON_KEY.
